# 🛒 Toko Online PHP Native

Project website toko online sederhana berbasis PHP Native dan MySQL yang dibuat untuk pembelajaran CRUD, koneksi database, dan pengelolaan produk menggunakan XAMPP.

---

## 📌 Fitur

- Menampilkan daftar produk
- Detail produk
- Halaman tentang kami
- Halaman kontak
- Koneksi database MySQL
- Tampilan sederhana dan responsive
- Menggunakan PHP Native tanpa framework

---

## 🛠️ Teknologi yang Digunakan

- PHP Native
- MySQL
- HTML5
- CSS3
- Bootstrap
- XAMPP
